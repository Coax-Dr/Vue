# 单文件组件语法规范(SFC)
## 每一个vue单文件组件(.vue文件)由三部分组成：template、script、style

### template块
每一个*.vue文件最多同时包含一个顶层template标签 
template中的内容会被提取传递到@vue/compiler-dom进行处理

### script块
每一个*.vue文件最多可同时包含一个script标签
script标签内容进已ES module来执行

### script setup块
每一个*.vue文件最多可同时包含一个script setup标签
script setup会作为组件的setup函数使用，也就是说它会在每一个组件实例中执行，script setup顶层绑定的变量、函数会自动暴露给模板template

### style块
每一个.vue文件可以包含多个style标签
style标签可以通过scoped将样式封装在当前组件内
