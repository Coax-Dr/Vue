# 根组件 传递给createApp的选项用于配置根组件，当应用挂载时，作为渲染的起点
```
const RootComponent = {
    /* 配置根组件的选项 */
}
const app =  Vue.createApp(RootComponent);
// 将app根组件挂载到id为app的标签下，mount返回根组件实例
const vm = app.mount("#app");
```
# 组件实例属性 在data中定义的属性通过组件实例暴露
## 组件实例的属性包括：methods，props，computed，inject，setup
```
const app = Vue.createApp({
    data() {
        return { count: 4 }
    }
});
// mount返回组件实例
const vm = app.mount("#app");
// 通过组件实例获取属性
console.log(vm.count);
```
