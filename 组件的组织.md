# 组件系统 几乎所有应用界面都是由小型、独立、可复用的组件构成

# 可复用的实例
 组件与根组件相同，都具有data、computed、watch以及生命周期钩子函数
 多次复用的任意组件之间都是相互独立的，都是新的实例，数据状态相互不影响

# 组件的注册
 为了能让Vue识别，在使用组件之前，首先需要对组件进行注册
## 全局注册 注册之后的组件可以用在任何新创建的组件实例的模板中
```
const app = Vue.createApp({});
app.component("component-a", {
    ...
});
app.component("component-b", {
    ...
});
app.mount("#app");

<div id="app">
<component-a></component-a>
<component-c></component-c>
</div>
```
## 局部注册
```
// 局部组件
import ComponentA from "./ComponentA.vue";
import ComponentB from "./ComponentB.vue";
export default {
    components: {
        // 在局部组件注册子组件
        ComponentA,
        ComponentB
    }
}
```
